# Assignment 6 – Medians and Order Statistics & Data Structures

## Files
- selection_algorithms.py
- data_structures.py

## How to Run

Run selection algorithms:

python selection_algorithms.py

Run data structures demo:

python data_structures.py

## Summary

This assignment implements:

Part 1:
- Randomized Quickselect
- Deterministic Median of Medians

Part 2:
- Arrays
- Stacks
- Queues
- Linked Lists

Performance analysis shows that both selection algorithms achieve linear time complexity, though randomized selection performs faster in practice.
